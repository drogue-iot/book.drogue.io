Pre-requisites

Deploying Drogue Cloud, you will need a few tools on your local machine. In most cases a Linux/Unix environment is assumed, which includes Mac OS X.

Command line tools

  • drg- aka Drogue command line client

  • http - aka HTTPie

  • mqtt - aka MQTT CLI

  • kubectl - https://kubernetes.io/docs/tasks/tools/install-kubectl/
